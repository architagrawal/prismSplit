# PrismSplit Brand Guide

## Brand Identity

### Mission
Make bill splitting effortless, transparent, and fair for friend groups everywhere.

### Visual Identity

#### Logo
- Place logo files here (SVG, PNG in different sizes)
- Logo variations: full color, monochrome, icon only

#### Color Palette

**Primary Colors**
- Primary: `#007AFF` (iOS Blue)
- Secondary: `#5856D6` (Purple)

**Semantic Colors**
- Success: `#34C759` (Green)
- Danger: `#FF3B30` (Red)
- Warning: `#FF9500` (Orange)
- Info: `#007AFF` (Blue)

**Neutral Colors**
- Background Light: `#FFFFFF`
- Background Dark: `#000000`
- Surface: `#F2F2F7`
- Text Primary: `#000000`
- Text Secondary: `#3C3C43` (60% opacity)
- Border: `#C6C6C8`

#### Typography
- **Primary Font**: System (SF Pro on iOS, Roboto on Android, Inter on Web)
- **Weights**: Regular (400), Medium (500), Semibold (600), Bold (700)

**Type Scale**
- Heading 1: 34px / Bold
- Heading 2: 28px / Bold
- Heading 3: 22px / Semibold
- Body: 17px / Regular
- Caption: 15px / Regular
- Small: 13px / Regular

#### Spacing
- Base unit: 4px
- Scale: 4, 8, 16, 24, 32, 48, 64

#### Border Radius
- Small: 8px (buttons, inputs)
- Medium: 12px (cards)
- Large: 16px (modals, sheets)
- Extra Large: 24px (hero elements)

#### Shadows
- Small: `0 1px 3px rgba(0,0,0,0.12)`
- Medium: `0 4px 6px rgba(0,0,0,0.16)`
- Large: `0 10px 20px rgba(0,0,0,0.19)`

### Iconography
- Style: Lucide Icons (outline style)
- Size: 20px, 24px, 32px
- Stroke width: 2px

### Voice & Tone
- **Friendly**: Use conversational language
- **Clear**: Avoid jargon, be direct
- **Helpful**: Guide users through complex flows
- **Positive**: Focus on fairness and friendship

### Example Copy
- "Split the bill, not friendships"
- "Fair splits made simple"
- "Everyone pays their share"
- "Add items and we'll do the math"

## Usage Guidelines

### Do's
✅ Use consistent spacing and alignment  
✅ Maintain adequate color contrast (WCAG AA minimum)  
✅ Use semantic colors appropriately  
✅ Keep UI elements properly sized for touch targets (44x44px minimum)  
✅ Provide clear feedback for user actions  

### Don'ts
❌ Don't mix different icon styles  
❌ Don't use brand colors on backgrounds without checking contrast  
❌ Don't use too many colors in one view  
❌ Don't make text smaller than 13px  
❌ Don't use animation just for decoration
