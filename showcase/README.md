# Screenshots & Demo

This folder contains visual assets for showcasing PrismSplit.

## Structure

### `/screenshots/`

App screenshots for app stores and README

- Mobile (iOS and Android)
- Web version
- Key features demonstrations

Naming convention:

- `ios-home.png`
- `android-split-bill.png`
- `web-dashboard.png`

### `/releases/`

Assets for release announcements

- Feature highlight images
- Release notes graphics
- What's new screens

### `/web/`

Assets for landing page and marketing

- Hero images
- Feature demonstrations
- Social media cards (Open Graph, Twitter Cards)

## Guidelines

### Screenshot Specifications

**Mobile Screenshots**

- iOS: 1290 x 2796 px (iPhone 15 Pro Max)
- Android: 1080 x 2400 px (Pixel 6)

**Web Screenshots**

- Desktop: 1920 x 1080 px
- Tablet: 1024 x 768 px

### Best Practices

- Use actual app content (not lorem ipsum)
- Show realistic data
- Ensure good lighting/contrast
- Remove any personal information
- Include both light and dark mode variants

### Tools

- **iOS Simulator**: Cmd+S to capture screenshot
- **Android Emulator**: Use capture button in toolbar
- **Web**: Browser screenshot tools or Figma exports

### Editing

- Use consistent device frames (if any)
- Add captions if needed
- Compress images for web (use TinyPNG or ImageOptim)
- Use WebP format when possible for better compression
